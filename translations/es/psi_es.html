Guía de instalación para el paquete de idioma español

Notas generales

Descarga el paquete de idioma haciendo click en el enlace "Psi translation - Español" en la parte superior de esta página.
Coloca el archivo en el directorio adecuado para tu sistema operativo según lo indicado en las instrucciones de más abajo.
El idioma se establece automáticamente utilizando la configuración de idioma de tu sistema. Si quieres cambiar el idioma en Psi manualmente, puedes hacerlo en la opción "Cambiar perfil".

GNU/Linux

Instalación como usuario

Copia psi_es.qm en $HOME/.config/psi/.

Mac OS X

Instalación para todo el sistema

Copia psi_es.qm en Aplicaciones/Psi.app/Contents/Resources/.
Para hacerlo, pulsa con el botón derecho del ratón (o Control+click) en Psi en la carpeta de aplicaciones, y a continuación haz click en "Mostrar contenido del paquete". Entra en la carpeta "Contents" y después "Resources". Entonces coloca tu psi_es.qm aquí.

Windows 2000/XP/Vista

Instalación como usuario

Copia psi_es.qm en %APPDATA%\Psi.

Instalación para todo el sistema

Copia psi_es.qm en C:\Archivos de programa\Psi. (Si instalaste Psi en otro lugar, copialo ahí.)
